Task 5.4: Build Release APK
  Status: NOT STARTED
  Time: 10-15 minutes
  Command:

  Outcome: Production APK file ready

  Status: NOT STARTED
  Time: 10 minutes (one-time)
  Command:

  Outcome: Signed APK ready for distribution

   - Keep terminal window open with "npm start"

   - Need it every time you rebuild
